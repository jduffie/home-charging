# charger — Emporia Pro

Device-specific configuration for the Emporia Pro home charger lives here once it's
worked out: pairing, app/account setup, any home-automation or load-management
hookup, and the knobs unique to this hardware. This folder owns only what is
concrete and specific to this device — not general advice. Situational
sizing/install questions ("what amperage?", "do I need load management?", "hardwired
vs plug-in?") are **not** answered in prose here; they depend on your home and panel
— use the copy-paste prompt instead:
[../prompts/charger-sizing.md](../prompts/charger-sizing.md).
