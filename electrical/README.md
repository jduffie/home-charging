# electrical

Notes on the home electrical side of charging: circuit and wiring as installed,
install estimates, and panel + load specifics for this house. This folder owns
only what is concrete and specific to the install — not general advice.

Situational install/sizing questions ("what amperage do I need?", "what wire
gauge?", "do I need load management?") are **not** answered in prose here — they
depend on your home, panel, and local code. Use the copy-paste prompt instead:
[../prompts/charger-sizing.md](../prompts/charger-sizing.md).
